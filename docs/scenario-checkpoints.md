# Scenario Checkpoints

Para pruebas manuales y automatizadas, verificar estos flujos (Checkpoints):
1. Login como invitado (`/guest` o `/guest2`).
2. Creación de torneo (verificar asignación de admin y slug).
3. Carga de predicciones (verificar autoguardado).
4. Configuración del torneo (cambiar reglas y verificar recálculo automático).
5. Sincronización multi-torneo (copiar predicciones).
