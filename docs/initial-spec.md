# Initial Spec

**Prode Mundial** nace como una aplicación web para organizar torneos de predicciones de resultados deportivos (World Cup 2026, Champions League).

Requerimientos iniciales:
- Multitorneo y multiusuario.
- Puntuación personalizable (Posiciones y Partidos).
- Soporte multilingüe (11 idiomas).
- Avatares personalizados y rankings dinámicos.
