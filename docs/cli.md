# Comandos CLI Útiles

## Desarrollo Frontend
- `npm run dev`: Inicia el servidor de desarrollo.
- `npm run build`: Construye la app para producción.

## Supabase CLI
- `npx supabase start`: Inicia el entorno local de Supabase.
- `npx supabase edge-functions serve sync-ucl-results`: Prueba la edge function localmente.
- `npx supabase db push`: Sube cambios de esquema.
