# Modding / Customization Reference

La personalización de torneos (modding) incluye:
- **Puntuación**: Ajustes en `tournament_config` (pts por ganador, empate, resultado exacto, diferencia de gol).
- **Avatares**: Asignación de avatares especiales (animales, clubes, famosos).
- **Reglas de Desempate**: Configuración paramétrica del descuento por empate fallido.
